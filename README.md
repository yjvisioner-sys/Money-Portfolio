# 📊 포트폴리오 대시보드

실시간 자산 현황 관리 + 경제 뉴스 수집 Streamlit 앱

## 🚀 설치 및 실행

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 열기

---

## 📦 주요 기능

| 기능 | 설명 |
|------|------|
| **실시간 시세** | yfinance를 통해 한국/미국 주식 실시간 조회 |
| **포트폴리오 KPI** | 총 평가금액, 투자금액, 총 손익, 종목 수 |
| **자산 배분 차트** | 유형별/통화별 파이차트, 종목별 비중 막대 |
| **경제 뉴스** | 글로벌 RSS 피드 자동 수집 (Reuters, FT, CNN Money 등) |
| **주요 지수** | KOSPI, KOSDAQ, S&P500, 나스닥, 다우, 니케이 |
| **자산 관리** | 추가/수정/삭제, CSV 내보내기 |

---

## 🗂️ 지원 티커 형식

- **한국 주식**: `005930.KS` (삼성전자), `000660.KS` (SK하이닉스)
- **미국 주식**: `AAPL`, `NVDA`, `TSLA`
- **ETF**: `069500.KS`, `SPY`, `QQQ`
- **현금**: `CASH_KRW`, `CASH_USD` (자동 처리)
- **지수**: `^GSPC`, `^KS11` 등

---

## 🔧 커스터마이징

`app.py` 상단 `st.session_state.assets` 초기값을 수정하여 기본 포트폴리오 변경 가능

---

## ⚠️ 참고사항

- 한국 주식은 장 마감 후 전일 종가 기준
- 환율(USD/KRW)은 5분 캐싱
- 뉴스는 10분 캐싱
- 네트워크 상태에 따라 일부 데이터 지연 가능
